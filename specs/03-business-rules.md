# Business Rules

## Rating
- Scale 0–5, 0.5 increments (10-icon interface: 5 large = 1pt, 5 small = 0.5pt)
- Rating required, review optional (500 char max)
- One log per user per show, editable, not duplicable

## Wishlist vs Playlist
- Wishlist: always private, only for non-logged, non-finished shows
- Playlist: show must be logged first; if not, prompt to log before adding
- Logging a wishlisted show auto-removes it from wishlist

## Profiles & Privacy
- Public profile: journal, favorites, public playlists visible to all
- Private profile: visible only to approved followers
- Wishlist: always private regardless of profile visibility
- Stats: visible to the user only, never to others
- Follow public profile: instant. Follow private profile: requires approval

## Moderation (mandatory MVP)
- Report content (posts, reviews)
- Block user
- Admin can remove content
- Required for App Store compliance (UGC guideline)

## Comments
- Excluded from MVP. Do not build unless explicitly requested.
