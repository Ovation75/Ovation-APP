# MVP Scope

## In scope (MVP)
- Onboarding & signup (Apple / Google / Email)
- Feed (community activity + show discovery + editorial picks)
- Découverte (trending, new, 4 genre categories)
- Search (shows / people)
- Mon Carnet: Journal (log + rate), Playlists, Wishlist (always private)
- Fiche spectacle (show detail page)
- User profile (public or private)
- Notifications
- Settings
- UGC moderation: report content, block user, admin removal (mandatory for App Store approval)
- Minimal admin workflow: catalogue management, weekly status updates, editorial picks, moderation queue

## Out of scope (v1)
- Comments (excluded unless explicitly re-added later)
- Ticketing link integration
- Map view
